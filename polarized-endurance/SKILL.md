# Polarized Endurance Training

## 한 줄 요약

**훈련 시간의 ~80%를 쉬운(Z1~Z2) 강도로, ~20%를 아주 힘든(Z4~Z5) 강도로.** 어중간한 "중간 강도(Z3, tempo/threshold 장시간)"는 피로 대비 적응이 낮다 — **피로-적응 비율의 트레이드오프**.

## 증거 강도

- 엘리트 지구력 선수 실제 분포: **강함** (관찰 연구 다수, Seiler, Esteve-Lanao, Tønnessen)
- 폴라라이즈드 vs 역치 중심 RCT: **보통** (크로스오버 연구 혼재, 효과 크기 ±)
- 레크리에이션~중급자에서도 유사 이득: **보통** (Stöggl & Sperlich 2014 등)
- 정확한 80/20 비율의 마법: **약함** (75~90/10~25 구간 비슷)

## 이론·증거 기원

- **Stephen Seiler** (노르웨이) — 엘리트 크로스컨트리·보트·사이클 분포 연구
- **Esteve-Lanao, Seiler, Foster** — 장거리 러너 분포와 결과
- **Stöggl, T. & Sperlich, B.** (2014, 2015) — "Polarized training has greater impact..." Frontiers in Physiology
- **MAF method (Maffetone)** · **Z2 강조 (Iñigo San Millán · Peter Attia)** — 미토콘드리아 기능과 대사 유연성

## 핵심 개념

### 1. 강도 구역 (5존 기준)

| 존 | 강도 | 대사 | 주관 | 호흡 |
|---|---|---|---|---|
| Z1 | Recovery | 지방 | 매우 편함, 대화 자유 | 코호흡 |
| **Z2** | Aerobic base | 지방+탄수 | 편함, 완전 문장 가능 | 입 살짝 |
| Z3 | Tempo | 혼합 | 말 짧게, 꾸준한 불편 | 거침 |
| Z4 | Threshold | 탄수 주 | 짧은 단어만 | 거침 |
| **Z5** | VO2max/Anaerobic | 탄수·무산소 | 말 불가 | 헐떡 |

### 2. Seiler의 3존 통합
- Zone 1 (low) = Z1+Z2 → **80%** of time
- Zone 2 (moderate) = Z3+threshold → **<10%**
- Zone 3 (high) = VO2max+ → **~20%**

### 3. 왜 중간 강도가 비효율?
- 피로 누적 빠름 vs 미토콘드리아 밀도·LT 향상은 Z2가 더 효율
- VO2max 향상은 Z5 짧은 간헐이 가장 강력
- Z3 중심은 "중간 정도 피곤한데 중간 정도밖에 안 늘어나는" 함정

### 4. FTP·LT·VO2max 주요 테스트
- **FTP(Functional Threshold Power)**: 1시간 지속 가능 최대 출력 (사이클)
- **LT1/LT2**: 혈중 젖산 역치
- **VO2max 테스트**: 실험실 또는 Ramp 테스트

### 5. 심박·출력 기반
- 심박: MaxHR·LTHR 기준 %
- 출력(파워미터): %FTP — 사이클에서 정확도 가장 높음
- 러닝: 페이스·심박·RPE 혼용

## 언제 쓰나

- **지구력 목표**: 마라톤·울트라·그란폰도·철인·전국일주 라이딩
- **FTP·VO2max 정체** — 중간 강도에 묶여 있을 때
- **주 5h+ 가용 시간** (주 3h 이하는 다른 분배 유효)
- **오버리칭·번아웃 이력** — 저강도 베이스 재구축
- **대사 건강 목적** — Z2 중심으로 주 3~5h

## 실전 프로그램

### 레크리에이션 사이클 주 6~8h
```
Mon: 휴식
Tue: Z2 60~90min
Wed: VO2 intervals 4~5 x 4min @Z5 / 3min recov (총 60min)
Thu: Z2 60min
Fri: 휴식
Sat: Long Z2 2~4h
Sun: Z2 60~90min + 짧은 Z4 blocks 선택
```
→ 저강도 ~80%, 고강도 ~15~20%

### 러너 주 40~60 km
```
쉬운 페이스 런 3~4회 (대화 가능)
VO2 세션 주 1회 (400m x 8, 1km x 5~6)
롱런 주 1회 (Z2)
threshold 런은 월 1~2회 또는 생략
```

### VO2max 인터벌 레시피
- **4x4min @Z5** / 3min 회복 (Norwegian 4x4, 가장 잘 연구됨)
- **5x3min** / 3min
- **30/15** (Rønnestad 세션) — 13x30sec 최대 / 15sec Z1, 3세트
- **40/20** — 초보자

### Z2 식별법
- **코호흡 가능**한 최대 강도
- **대화 테스트**: 완전한 문장이 여전히 가능
- 심박: 대략 maxHR의 60~72% 또는 LT1 아래
- 사이클: 대략 FTP의 55~75%

## 측정·진행

- 주 TSS/CTL 추적 (TrainingPeaks·Intervals.icu·Zwift)
- 4~8주마다 FTP 또는 10k 페이스 테스트
- HRV·안정시 심박 (`recovery-periodization`과 연결)
- **VO2max percentile** (연령·성별 기준) — 고위험 <20th, 평균 40~60th, 우수 >80th. 전사망률 예측에서 가장 강력 (Mandsager 2018)

### VO2max 추정값 (연령·성별 대비 양호 구간, ml/kg/min)
| 연령 | 남(양호) | 여(양호) |
|---|---|---|
| 30대 | 44~51 | 38~44 |
| 40대 | 41~47 | 35~40 |
| 50대 | 37~43 | 32~37 |
| 60대 | 33~39 | 28~33 |
잘 훈련된 라이더·러너는 +10 이상. Cooper Institute 기반, 개인차 큼.

## 한국 사이클·러너 현실

- **한강·탄천·팔당** 등 도심 라이딩 코스: 신호·차량·정체. Z2 유지 어려움 → 한적한 시간대(새벽)·교외 코스 활용
- **실내 트레이너(Zwift·Wahoo·Tacx)** — 한국 주거 환경(아파트)의 진동·소음 문제. 발코니·별도 공간·고무매트. 구조화된 Z2 유지에 유리
- **그란폰도·전국일주** — 장시간 Z2 지속력이 핵심. 5~8h 안장 적응은 훈련 + 장비(새들·비브)로 분리 관리
- **러너의 관절 부담** — 한국 아스팔트·시멘트 비율 높음. 주 1회 잔디·흙길 또는 수영·사이클 크로스 권장
- **여름 35℃+ / 겨울 -5℃ 이하** — 극단 온도 시 강도 -10~15%, 수분·전해질 명시적 보충

## 여성·고령 특례

- **여성 지구력 선수**: 철 결핍·RED-S 리스크 특히 높음. 페리틴 <30 시 수행 저하 명확. 6개월 1회 CBC + 페리틴 권장
- **고령자(50세+)**: Z2 주 3~5h + VO2 주 1회로 압축. 회복 72h+ 필요 가능. 관절 부담으로 사이클·수영 우위
- **폐경기**: 근감소·골밀도 대비 저항훈련 비중 유지. 달리기만으로는 골다공증 방어 부족

## 안티패턴

- **매일 Z3 "gray zone"** — 늘 피곤한데 늘지 않음
- **Z2를 너무 빠르게 하는 것** — 심박 측정 없이 "편한" 기준이면 Z3로 흘러감
- **VO2 세션 매주 2회+ 장기** — 누적 피로, 면역 저하
- **Long ride 매 주말 Z3 drift** — 시작은 Z2인데 끝엔 Z3
- **80/20을 단일 훈련 내에서 비율 맞추기** — 주간·블록 단위 분포 얘기
- **근력 훈련 생략** — 지구력만으로는 근감소 예방 부족. 저항운동 병행 권장
- **고탄수 Z2는 지방 적응을 망친다 주장** — 대부분의 선수에게 해당 없음. 탄수 보충은 긴 세션에서 필수
- **심박 센서 부정확 운용** — 옵티컬보다 흉부 스트랩 우선, 특히 고강도

## 한계

1. **시간 대 효과** — 주 3h 이하에선 "THR 중심" 또는 혼합이 더 효율일 수 있음
2. **VO2max 개인차 큼** — 반응자 스펙트럼
3. **종목 특이성** — 저강도 러닝 volume은 관절 부담 → 러너는 크로스 트레이닝 필요
4. **연령·회복 능력**에 따라 주 고강도 세션 2회 → 1회 조정
5. **혈중 젖산 측정 부담** — 휴대형 기기 있으나 실용 한계
6. **대회 특이성** — 그란폰도·크리테리움·트랙 경기의 공인 프로토콜은 다를 수 있음

## 함께 쓰는 것·이 프레임워크가 *틀렸을 때*

- 회복·주기화 → `recovery-periodization` (HRV, deload 주)
- 근력 병행 → `strength-basics` (주 2회 하체 최소)
- 체중 관리 → `macro-tracking` (훈련량 대비 적자 과하면 수행 급락)
- 식단 질 → `mediterranean-diet`
- 심혈관·혈액 모니터 → `blood-biomarkers`
