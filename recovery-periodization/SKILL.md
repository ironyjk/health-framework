# Recovery & Periodization

## 한 줄 요약

**적응은 훈련이 아니라 회복 중에 일어난다.** 자극·회복·적응(SRA) 곡선을 이해하고 주기화(periodization)로 의도적 축적·감소를 반복해야 장기 progression이 유지된다.

## 증거 강도

- SRA 모델: **강함** (Zatsiorsky, Matveev, Fitness-Fatigue 모델)
- Deload 주기의 이득: **보통~강함** (실증 직접 RCT는 적으나, 코칭 경험·고급자 데이터 일치)
- HRV와 훈련 적응 상관: **보통** (개인 내 추세에 유용, 절대값 비교는 제한)
- 안정시 심박 증가 = 과로 신호: **보통**
- 과훈련 증후군 생리 지표: **보통~혼재** (임상 진단 기준은 여전히 느슨)
- "콜드플런지·사우나 = 회복 가속" 주장: **약함~혼재** (특정 문맥 유효)

## 이론·증거 기원

- **Hans Selye** — General Adaptation Syndrome (Alarm-Resistance-Exhaustion)
- **Yakovlev** — 초과회복(supercompensation) 곡선 (단순화된 모델, 한계 있음)
- **Zatsiorsky & Kraemer** — Fitness-Fatigue 2요인 모델 (더 정확)
- **Matveev** — 주기화 원형 (Classical Periodization)
- **Issurin** — Block Periodization
- **Tudor Bompa** — *Periodization: Theory and Methodology of Training*
- **Kellmann, Meeusen et al.** — Overreaching/Overtraining 합의 (2013 Euro J Sport Sci)

## 핵심 개념

### 1. SRA 곡선
```
Stimulus → Fatigue (수행 저하) → Recovery (기저 복귀) → Adaptation (기저 초과)
```
자극 강도가 크면 Fatigue 깊고 Recovery 오래. 다음 자극은 보통 **Adaptation 상승 구간**에서 주는 것이 이상.

### 2. Fitness-Fatigue 2요인 모델
- Performance = Fitness - Fatigue
- Fitness: 천천히 쌓임·천천히 사라짐
- Fatigue: 빠르게 쌓임·빠르게 사라짐
- **Taper**: 며칠 낮은 훈련량 → Fatigue↓ > Fitness↓ → 수행 최대

### 3. 주기화 형태
- **Linear**: 볼륨↓ · 강도↑ (고전 블록)
- **Undulating (DUP)**: 주/일 단위 강도·반복 변동
- **Block (Issurin)**: 2~4주 축적 → 1주 변환 → 1주 피크 반복
- **Conjugate (Westside)**: 최대·다이나믹·반복 일 혼합

### 4. Deload
- 4~6주 accumulation 후 1주 volume -40~60%, intensity -10%
- Neural deload: volume 보존, intensity -20%
- 고급자일수록 더 필요

### 5. 오버리칭 vs 오버트레이닝
| 상태 | 기간 | 회복 | 특징 |
|---|---|---|---|
| Functional Overreaching | 수일~주 | ~2주 내 수행 반등 | 의도된 축적 |
| Non-functional Overreaching | 수주 | 수주~수개월 | 의도 없는 과부하 |
| **Overtraining Syndrome** | 수개월+ | 수개월~년 | 내분비·면역·심리 광범위 |

### 6. 오버트레이닝 vs 번아웃 — 구분 플로차트

운동 과훈련(Overtraining Syndrome)과 직장·심리 번아웃(Maslach Burnout)은 증상이 **광범위하게 겹친다**. 지속 피로·수면 악화·기분 저하·동기 상실은 둘 다 같은 그림. 잘못 구분하면 대처도 반대로 간다(과훈련엔 쉼, 번아웃엔 환경·의미 재설계).

**1차 구분 질문 — 3개 중 2개 이상 "예" → 과훈련 가능성 우선:**
- 지난 4~8주 훈련 볼륨·강도가 평소 대비 ↑
- 수행 지표 하락이 객관 측정(1RM·파워·타임)으로 확인됨
- 휴가·여행 등 업무에서 완전히 분리된 주에도 피로·수행 저하 지속

**1차 구분 질문 — 3개 중 2개 이상 "예" → 번아웃 가능성 우선:**
- 업무·관계에서 냉소·거리감·무가치감 느낌이 주(week) 단위로 지속
- 휴가·주말에 일에서 벗어나면 급격히 회복됨 (며칠 만에)
- 운동은 평소와 같거나 줄었는데도 피로감이 존재

**공통 영역(둘 다 해당):**
- HRV 하락, RHR 상승, 수면 분절 — 생리 지표는 둘 다 비슷하게 나옴
- 감염 빈발, 상처 회복 느림
- 식욕·체중 변화
- 이 경우 **생활 전반의 allostatic load 재조정 + 전문의 평가** 우선

**Maslach 3축 (번아웃 진단 보조):**
| 축 | 설명 | 과훈련과의 차이 |
|---|---|---|
| Exhaustion | 정서·신체 고갈 | 과훈련과 증상 겹침 |
| Cynicism | 일·사람에 대한 거리감·냉소 | **번아웃 고유** — 과훈련엔 거의 없음 |
| Inefficacy | 성취감 상실·"내가 쓸모없다" | **번아웃 고유** — 과훈련은 수행 저하는 있으나 자기평가 왜곡은 덜함 |

**대처 방향 차이:**
- 과훈련 → 훈련 볼륨 40~60% 감축, 2~4주 deload, 필요 시 1~3개월 중단. 이 프레임워크 내부에서 해결 가능.
- 번아웃 → 훈련 축소만으론 해결 안 됨. 업무 경계·의미·통제감·동료 지원 등 환경 재설계 필요. `counsel` 스킬 또는 전문가 상담 권고. 운동은 유지하되 "즐거움 위주"로 전환(고강도 훈련 일시 축소).

**오진 시 리스크:** 번아웃 환자에게 "더 쉬어라"만 처방하면 고립·기분 저하 악화. 과훈련 환자에게 "정신 문제"로 돌리면 내분비·면역 손상이 장기화.

## 회복 지표

### 객관
- **HRV (심박 변동성)** — 아침 평균, 개인 baseline 추세. 7일 rolling vs 28일
- **RHR (안정시 심박)** — 기저 +5bpm 이상 지속 = 신호
- **체중 + 식욕** — 급락 추세
- **수면 질·효율**
- **수행 지표** — 동일 부하 RPE·심박 상승

### 주관
- **POMS·RESTQ-Sport** 설문
- "오늘 훈련 기대되는가" 간단 평가 1~10
- 컨디션 일기 (3~5분/일)

### 경고 사인 조합
- 주 세션 수행이 4세션 연속 RPE 상승 + 동일 중량
- RHR +7~10bpm 1주 이상
- 수면 악화 + 기분 저하
- 반복 경미 부상·감염

## 실전 적용

### 저항운동 중급 (5/3/1 사이클)
```
W1: 5/5/5+ @65/75/85%
W2: 3/3/3+ @70/80/90%
W3: 5/3/1+ @75/85/95%
W4: Deload 5/5/5 @40/50/60%
```

### 근비대 블록 (RP)
```
4~6주 accumulation (MEV → MRV)
1주 deload (volume 50%, intensity 유지)
반복 또는 강도 블록 전환
```

### 지구력 주기화
```
Base 8~12주 (Z2 주 ~80%)
Build 4~8주 (VO2 ↑, 임계)
Peak·Taper 1~3주
Race
Transition 2~4주 (회복, 볼륨 낮음)
```

### 일상 회복 도구
- **수면**: 가장 강력. `sleep-foundations` 참고
- **단백질·탄수 충분**: `macro-tracking`
- **스트레스 관리**: 명상·마음챙김 (증거 보통)
- **가벼운 유산소(Z1) 다음 날**: active recovery, 혈류 촉진
- **사우나**: 열순응·심혈관 이점 (보통 증거, Laukkanen 2015), 회복 가속은 약함
- **콜드플런지**: 근력 훈련 직후엔 **근비대 저해** 가능 (Roberts 2015). 지구력엔 중립~이익 가능. 토너먼트·연속 경기일 사이 수행 유지엔 유효
- **마사지·폼롤링**: 주관 회복·통증감 감소 (ROM 효과 한정)

### 여성 생리주기 회복 조정

- **난포기 (생리 시작 ~배란)**: 에스트로겐 상승 — 회복력 상대적 우수, PR·고볼륨 유리
- **배란~황체 초기**: 안정적
- **황체기 (생리 전 5~7일)**: 체온·심박·수면 분절 증가. 같은 RPE에 수행 ±5~10% 하락 가능. 볼륨 유지하되 RIR 여유 / 스트레스 큰 주는 deload로 이동
- **생리 기간**: 주관 불편·철 손실. 가벼운 활동 선호, 본인 반응 우선
- **RED-S 위험**: 3개월+ 무월경 + 에너지 가용성 <30 kcal/kg FFM = 훈련 중단 및 의료 평가
- 데이터: FitrWoman·Clue 등 앱 + 훈련 로그 교차

### RED-S 체크리스트 (IOC 2018 기반)

다음 ≥3개면 RED-S 의심, 전문의 평가:

- 3개월+ 무월경 또는 월경 주기 불규칙
- 체중·체지방 지속 감소 또는 저체중(BMI <18.5)
- 수행 저하(동일 훈련에 이전보다 현저히 힘듦)
- 피로골절·과사용 부상 반복
- 감염·감기 빈번
- 기분 저하·집중 저하·수면 악화
- 안정시 심박 이례적 저하(과훈련성)·레이노 증상
- 식이 제한·보디이미지 강박 징후
- 변비·장 기능 저하
- **남성도 발생** — 아침 테스토스테론 저하·성욕 감소 신호

### 훈련 외 allostatic load (합산)

적응 예산은 훈련만의 몫이 아니다. 다음을 합산 관리:

- **직장 스트레스** (데드라인·야근·갈등)
- **가족** (어린 자녀·간병·관계 위기)
- **감염·예방접종 후 48~72h**
- **시차·출장**
- **소음·공해 환경**
- **재정·법적 스트레스**

총부하가 높은 주에는 훈련 볼륨 -20~30% 선제 조정 (사후 반응 아님).

## 안티패턴

- **Deload 생략 장기** — "약해진다" 공포, 실제로는 퇴행
- **피로·수행 저하에 볼륨 더 투여** — 가장 흔한 실패
- **HRV 하루 값에 반응** — 주/주-대-주 추세가 지표
- **단일 도구 만능 의존** (아이스배스·사우나·보충제) — 수면·영양 앞자리
- **"기분 나쁜 날은 쉰다" 무원칙** — 훈련 적응 불안정
- **대회 전 피로 남겨두고 taper 부족** — 수행 저하
- **훈련 외 스트레스(일·가족·수면·감염) 무시** — 총 allostatic load는 합산
- **"더 많이 = 더 좋음" 선형 사고**

## 한계

1. **Supercompensation 단순 모델의 한계** — 실제는 다중 타임스케일 적응
2. **HRV 해석 난이도** — 아침·기립·주기·호흡에 민감. 기기 편차
3. **오버트레이닝 진단 기준 느슨** — 배제 진단 성격, 시간 오래 걸림
4. **개인차** — 같은 프로그램이 누군가엔 과소, 누군가엔 과다
5. **훈련 외 인생 요인**이 큰 변수 — 통제 불가 영역 존재
6. **회복 도구 마케팅 과잉** — 기본기(수면·영양)만큼 효과 주는 가젯은 거의 없음

## 함께 쓰는 것·이 프레임워크가 *틀렸을 때*

- 수면 근본 문제 → `sleep-foundations`
- 영양 적자 과대 → `macro-tracking` 재설정
- 혈액 이상 의심 → `blood-biomarkers` (갑상선·철·비타민D·테스토스테론)
- 저항운동 과부하 설계 → `progressive-overload` · `hypertrophy-volume`
- 지구력 강도 분포 오류 → `polarized-endurance`
